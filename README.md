# Gerso Personalizados API

Backend API built with ASP.NET Core and Entity Framework Core for managing
orders, payments and order summaries for a small personalization business.

## Features
- Orders and Order Items
- Payments with partial payments (abonos)
- Order balance calculation
- SQL View integration (Order Summary)
- REST API with Swagger

## Tech Stack
- ASP.NET Core (.NET 8)
- Entity Framework Core
- SQL Server Express
- Swagger / OpenAPI

## Endpoints
- GET /api/orders/summary
- POST /api/payments
- GET /api/customers

## Notes
This project was built as a real-world backend example, not a tutorial demo.
